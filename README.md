# ChefSFTP
=======

# sftp-cookbook

Installs & Configures SFTP and SFTP users.

## Supported Platforms

Centos 7

## Usage

### sftp::default

Include `sftp` in your node's `run_list`:

```json
{
  "run_list": [
    "recipe[sftp::default]"
  ]
}
```
